The main purpose of Bootstrap is its grid system which helps to format your website layout

A row defines a horizontal bit of content Each row class typically has columns classes within which define vertical content A column can have varying sizes from 1 to 12. Columns can be defined without a size component so that each be of equal width Bootstrap also automatically makes the website responsive to the width of your browser screen.
