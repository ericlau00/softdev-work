What is Bootstrap?

A demonstration of Bootstrap 4 by Eric Lau and David Xiedeng.
The main purpose of Bootstrap is its grid system which helps to format your website layout

Everything is held within a container class which adds some margins and padding to the side of the website.
A row defines a horizontal bit of content.
Each row class typically has column classes within it which define vertical content.
A column can have varying sizes from 1 to 12.

  <div class="row">

    <div class="col-2"></div>
    <div class="col-3"></div>
    <div class="col-6"></div>
  </div>
</div>

One of the most important aspects of Bootstrap is the ability to respond to changes in screen size. If you shrink the browser size and look at the columns in the previous sections, you notice that the reading experience is greatly hindered. This can be fixed with the following
Bootstrap has 5 definitions of screen size:

  • xs: smaller than 576px
  • sm: 576px-768px
  • md: 768px-992px
  • lg: 992-1200 pixels
  • xl: larger than 1200px

The size of a col when the screen falls within one of the sizes can be defined with the following modifier:
 <div class="col-<screen-size><width-size>"></div>

A div can have multiple size definitions that are separated by a space such that the size changes when the screen size changes from smaller to larger. The following columns are defined as follows. Note that when the screen is shrunk to small, the columns takes up all 12 dividers. However, when on the large screen, the first column takes up 4 while the second takes up 8.
<div class="row">
  <div class="col-sm-12 col-lg-4"></div>
  <div class="col-sm-12 col-lg-8"></div>
</div>

Bootstrap also makes it super easy to work with margins and padding.
Margins and padding work on a scale of 1 to 5, 1 being the smallest and 5 being the largest.
<div class="row p<direction>-<size>">
 <div class="row m<direction>-<size>">
 directions include: t (top), b (bottom), l (left), r (right), x (left and right), y (top and bottom), or none (for all)

The Navbar gives a handy way to add a heading and searchbar to the top of your page. Elements in the navbar allows you to add clickable links without having and underline.

Link to Doc page

The navbar is started with the < nav> tag and is under the class "navbar." Padding to the bar can be added with class navbar-expand, with sizes -sm, -md, lg, -xl, or no size. Text colors in the nav bar is done with navbar-light or navbar-dark. Background color is done with bg-*, like -light, -dark, and -primary.

The navbar-brand class serves as the main logo, which is also clickable, like the globe on Wikipedia. These are basic link wrappers with the class navbar-brand: < a class="navbar-brand" href=> Text can be put between this wrapper for clickable text as the main logo. An image may be added with < img src="">, and text is still possible to be added.

The remaining clickable parts are put under an unordered list. The main < ul> tag has the class navbar-nav The list elements are given the nav-item class, and the links are nav-links.
Lists can also be entirely ignored with class tags "nav-item nav-link" for the links, and navbar-nav the main < div>. Dropdowns are slightly stricter with the structure. labeled by class="dropdown-menu" and verbatim , with each element labelled by dropdown-item.
